# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-21

### Added
- Initial release of reflex-sigma-graph
- Sigma.js integration for Reflex applications
- Support for interactive graph visualization
- Multiple layout algorithms (ForceAtlas2, Circular, Random, NoOverlap)
- Event handlers for node and edge interactions
- Customizable styling and theming
- Support for node and edge labels
- Demo application with sample usage

### Known Issues
- Requires manual npm package installation in `.web` directory
- Uses runtime file copying for component resolution
